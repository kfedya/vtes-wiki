# VTES Wiki Index

Last updated: 2026-04-17 | Pages: 28 | Sources: 1

Catalogue of every page in the wiki. LLM reads this first when answering a query to locate relevant pages, and updates it on every ingest.

## Conventions

- Entries are grouped by category (matching the folder structure).
- Each entry is one line: `- [[page-name]] — one-line summary (<150 chars)`.
- Keep summaries specific; they drive relevance decisions during retrieval.

## Rulings (primary)

### Turn structure
- [[game-overview]] — object of the game, turn sequence skeleton, setup, play area, important terms, Golden Rule, sequencing, wording templates.
- [[unlock-phase]] — unlock all cards, Edge gain, contested cards (1 pool/turn) and contested titles (1 blood/turn), burn-option cards.
- [[master-phase]] — master phase actions, out-of-turn masters and next-turn penalty, Red List mark.
- [[minion-phase]] — action order, mandatory hunt, "stuck" minions, per-card/per-minion limits, summary of an action.

### Actions
- [[bleed]] — base 1, directed at prey, 0 stealth; (limited) bleed modifiers; Edge only on successful ≥1; redirect reopens blocks.
- [[hunt]] — vampire-only, +1 stealth, undirected; mandatory for 0-blood vampires; excess over capacity drains.
- [[equip]] — from hand at listed cost, or move between your minions for free; announce which equipment at announcement.
- [[employ-retainer]] — +1 stealth; untransferable; retainers take environmental damage (cannot be dodged).
- [[recruit-ally]] — +1 stealth; recruited ally cannot act this turn (summoning sickness).
- [[leave-torpor]] — 2 blood, +1 stealth, undirected; if blocked by a vampire, the blocker gets a diablerie opportunity — no combat either way.
- [[rescue-from-torpor]] — 2 blood split cost; +1 stealth / undirected if same controller, 0 stealth / directed if different; blocked = combat.
- [[diablerise-action]] — action profile only; full resolution deferred to stage 3.
- [[become-anarch]] — untitled non-Anarch vampire only; 2 blood (or 1 with another ready Anarch); +1 stealth; undirected.

### Block & action resolution
- [[block-resolution]] — who may block, stealth/intercept comparison, add-only-when-needed, 3-state action course (A/B/C), wake effects, lock-on-success.

### Politics
- [[politics]] — umbrella: political action + referendum + voting + blood hunt.
- [[political-action]] — always undirected; referendum terms declared only on success; action success ≠ referendum pass.
- [[referendum]] — three steps (terms → polling → resolve); ties fail; blood-hunt referendum can't be blocked or reacted to.
- [[voting]] — title-vote table (Camarilla + legacy Sabbat/Laibon); Edge-for-1-vote; political-action-from-hand restrictions; locked OK, torpor abstains.

### Card types
- [[crypt]] — crypt cards (vampires): name, blood capacity, clan, Disciplines, group, advanced, text box.
- [[library]] — library cards umbrella: master vs minion, playing a card, drawing, requirements, targeting.
- [[master]] — master cards: regular vs out-of-turn, trifle, location, trophy, Discipline card.
- [[action]] — action cards: how a minion performs a non-default action; one per action; locks the minion.
- [[action-modifier]] — modify an ongoing action; no relock; can't repeat same modifier in a single action.
- [[reaction]] — response to another Methuselah's action; requires ready unlocked minion; does not lock.
- [[combat-card]] — card type used inside combat; combat mechanics ingest later (stage 3).
- [[political-action-card]] — political action card type; see the [[political-action]] ruling for action mechanics.
- [[ally]] — non-vampire independent minion with life from the blood bank; "as a vampire" rule.
- [[retainer]] — minion-on-minion with life; burned when employer burns or life hits 0.
- [[equipment]] — gear attached to the bearer; burned when bearer is burned.
