# Action Modifier Cards

## Summary
An **action modifier** is played by the **acting minion** to modify their action. It can be played **at any time before action resolution** [src-001].

## Core Rules
- Can only be played during an action (not outside of one).
- The acting minion is typically the one who plays the modifier, but some modifiers are explicitly played by **minions other than the acting minion** — in that case, only minions controlled by the **same Methuselah** as the acting minion can play them [src-001].
- A minion **cannot play the same action modifier card more than once** during a single action — even if using a different Discipline level [src-001].
- Action modifiers do **not** lock the minion playing them.

## Common Uses (examples only — not exhaustive)
- Add stealth to a bleed or directed action (e.g., Cloak the Gathering).
- Increase bleed amount (e.g., Govern the Unaligned's bleed-boost mode).
- Redirect an action to a different target.

Card-specific behaviour belongs in [[../../cards/]] pages when those cards become "problem cards".

## Cancellation
If cancelled, a non-action card's **cost is still paid as normal** [src-001]. The modifier's effects do not occur.
