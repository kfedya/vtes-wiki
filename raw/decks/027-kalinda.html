Highlights

This is a stealth & bleed starring Kalinda (G6) and relying on Haqim's Law: Retribution to increase its bleed, and Protected District for defense.

The idea is to influence Kalinda (G6), as her ability lets her bleed then unlock, to be ready to defend with Protected District.

Stealth is pretty high, with Obfuscate O modifiers doing their usual job. Swallowed by the Night and Resist Earth's Grasp are favored because they can be used for stealth or discarded by Haqim's Law: Retribution for more bleed power.

The combat module is mostly defensive, with a lot of maneuvers and some combat ends. But it can still surprise an unwary opponent with Hunger of Marduk and Pursuit.

Tips & Tricks

The deck packs a small political module that can help with the lunge, using Camarilla's Iron Fist. It can be negotiated witht he table, or pass thanks to the static votes available or Alamut, if it is available. Banu Haqim Justicar should be easy to pass and also provide more votes.

The Ashur Tablets & Infernal Pursuit combination is an efficient way to fix and improve the cards flow if needed.

The deck is close enough to Haqim Royalty that it can keep the opponents guessing for a while. It makes them cautious about entering combat with the vampires, seeing how that archetype is so strong in combat. It is a lever one should not hesitate to exploit when playing this deck.

Variants

Alamut can be doubled, and Web of Knives Recruit is a nice fit.
